# All-Pro Project Proof Intake

Use this checklist before publishing a city-specific project or case study. The goal is useful evidence without exposing a customer address or assigning a project to the wrong city.

## Required Project Record

| Field | What to record |
| --- | --- |
| Internal project name | Short service and city label, such as `Deck lighting - Belleville` |
| City | City only; do not publish the street address |
| City evidence | Estimate, invoice, work order, customer review, or project record that identifies the city |
| Service | Kitchen, bathroom, deck, fence, landscaping, concrete, patio, or small repair |
| Starting problem | What was damaged, outdated, unsafe, difficult to use, or missing |
| Scope | Work All-Pro agreed to perform |
| Work performed | Materials and major steps that can be supported by the record or photos |
| Outcome | Visible result or homeowner-reported result; label which one it is |
| Completion month/year | Month and year only unless a precise date is useful and approved |
| Photos | At least one before and one after photo when available |
| Photo permission | Confirm the homeowner allows public use and remove house numbers, faces, plates, and private documents |
| Review permission | Link to a public review or record written permission to publish a private testimonial |

## Photo Naming

Use factual filenames only after the city is confirmed:

- `service-city-il-before-01.webp`
- `service-city-il-after-01.webp`
- `service-city-il-detail-01.webp`

If the city is not confirmed, use `metro-east-il` instead of guessing.

## Case Study Format

1. Project scope
2. Problem addressed
3. Work performed
4. Visible or homeowner-reported outcome
5. Evidence/source note
6. Related service page
7. Estimate call to action

## O'Fallon Proof Needed

The current repository contains real Metro East project photos but no project record that safely confirms those photos were taken in O'Fallon. To publish the first true O'Fallon case study, provide:

- one O'Fallon estimate, invoice, or work order with the customer details kept private;
- at least one matching project photo;
- the service, starting problem, scope, and outcome; and
- confirmation that the photo may appear on the public website.

Until then, the public O'Fallon proof page must label the images as Metro East examples.
