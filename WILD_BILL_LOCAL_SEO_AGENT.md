# Wild Bill Local SEO Agent

This is the operating lane for scaling All-Pro local SEO without bloating the site with garbage pages.

## Goal

Use city + service pages, project proof, and internal links to win more high-intent searches in Metro East towns.

## Current Build

- Main landscaping page: `landscaping.html`
- City landscaping pages are generated by: `generate_local_seo_pages.py`
- The current generator covers:
  - `landscaping-belleville-il.html`
  - `landscaping-glen-carbon-il.html`
  - `landscaping-maryville-il.html`
  - `landscaping-troy-il.html`

## How To Run

From the repo root:

```powershell
python generate_local_seo_pages.py
```

The script will:

1. Regenerate the supported city pages.
2. Update `sitemap.xml` with any missing URLs.

## Rules

1. Do not mass-produce thin pages with only the city name swapped.
2. Every page should answer a real homeowner intent:
   - cleanup
   - mulch
   - drainage
   - grading
   - bed refresh
   - curb appeal
3. Every page needs:
   - city-specific headline
   - useful service bullets
   - strong CTA
   - internal links to related services
   - proof link to completed jobs or reviews
4. Use real nearby towns, not random stuffing.
5. Keep the money pages first:
   - Belleville
   - O'Fallon
   - Edwardsville
   - Fairview Heights
   - Collinsville
   - Glen Carbon
   - Maryville
   - Troy

## Weekly Cadence

1. Add 1 to 2 real project proof updates.
2. Add 1 new city/service page only if there is a gap.
3. Link the new page from the parent service page.
4. Confirm the page is in `sitemap.xml`.
5. Push photos and review asks through Google Business Profile separately.

- Do not create pages with fake neighborhoods or fake project stories.
- Do not rely on blogging alone when proof pages and reviews are the real gap.
