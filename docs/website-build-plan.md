# Website Build Plan

This document outlines the plan for building the contractor website.

---

## Goals

Create a simple, fast-loading contractor website that:

- Ranks in local search results
- Converts visitors into phone calls
- Supports Google Business Profile authority

---

## Pages to Build

- Home page (index.html)
- Services page (services.html)
- Contact page (contact.html)
- Individual city/service SEO pages

---

## Key Elements

Each page should include:

- Business name and phone number
- Clear list of services
- Service area cities
- Call to action (call for a free estimate)
- Google Business Profile link

---

## Technical Requirements

- Mobile-friendly layout
- Fast page load speed
- Meta title and description on every page
- Local keywords in headings and content
